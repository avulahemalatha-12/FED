Red Wine quality prediction

In this study, we propose a machine learning-based approach that utilizes clustering techniques for predicting red wine quality. We use a dataset that includes various attributes, such as acidity levels, residual sugar, sulfur dioxide levels, and alcohol content, which are commonly used to evaluate wine quality. The dataset is preprocessed to handle missing values, normalize data, and remove outliers.We apply the k-means clustering algorithm to group similar red wines based on their attribute values.
